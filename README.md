# Employee Schedule Manager Assignment

This repo contains two implementations of the employee scheduling application:

- `employee_scheduler.py` — Python version
- `EmployeeScheduler.java` — Java version

It also includes:

- `final_schedule_compact.png` — screenshot of a sample final schedule output
- `employee_schedule_submission.docx` — submission document (ready to upload)
- `employee_schedule_submission.pdf` — submission document (ready to upload)

## Run the Python version
```bash
python3 employee_scheduler.py
```

## Run the Java version
```bash
javac EmployeeScheduler.java
java EmployeeScheduler
```

Both versions support:
- sample data mode
- manual input mode
- ranked shift preferences (bonus)
- conflict handling
- readable weekly output
